# Schedule B: Interest and Ordinary Dividends

## Overview

Required if total interest or total ordinary dividends exceed $1,500, or if certain foreign account conditions apply.

## Who Files This

- Total taxable interest (1040 Line 2b) > $1,500
- Total ordinary dividends (1040 Line 3b) > $1,500
- Have a foreign bank account (FBAR/FinCEN 114 reporting)
- Received interest from a seller-financed mortgage

## Line-by-Line Instructions

### Part I: Interest (Lines 1-4)

#### Line 1: List each payer and amount
- **Source**: All 1099-INT Box 1, and any other interest income
- **Computation**: List each source separately with name and amount

#### Line 4: Total interest
- **Computation**: Sum of all amounts listed
- **Notes**: This should equal 1040 Line 2b

### Part II: Ordinary Dividends (Lines 5-6)

#### Line 5: List each payer and amount
- **Source**: All 1099-DIV Box 1a
- **Computation**: List each source separately

#### Line 6: Total dividends
- **Computation**: Sum of all amounts listed
- **Notes**: This should equal 1040 Line 3b

### Part III: Foreign Accounts and Trusts

#### Line 7a: Foreign accounts
- Did you have a financial interest in or signature authority over a financial account in a foreign country?
- If yes: may need to file FinCEN 114 (FBAR) separately

#### Line 7b: Foreign trust
- Did you receive a distribution from, or were you a grantor of, a foreign trust?

## Flows To

- **1040 Line 2b**: Part I total (interest)
- **1040 Line 3b**: Part II total (dividends)

## Notes

This is a simple listing schedule. The main purpose is to itemize sources when totals exceed $1,500 and to flag foreign account requirements.
